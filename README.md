# Integrated Financial Management System (IFMS)

## 📌 Project Overview
The Integrated Financial Management System (IFMS) is a web-based application designed to manage organizational financial assets efficiently. The system integrates asset valuation, depreciation, and lease accounting into a unified platform.

It provides automated calculations, real-time asset tracking, and role-based access control, making it suitable for Small and Medium Enterprises (SMEs), institutions, and organizations.

---

## 🎯 Key Features

- Unified Asset–Depreciation–Lease Management
- Automated Depreciation & Lease Calculations
- Dynamic Asset Value Recalculation
- Full Asset Lifecycle Tracking
- Role-Based Access Control (RBAC)
- RESTful API Architecture
- Web-Based Responsive Interface
- SME-Friendly Design

---

## 🏗️ System Architecture

The system follows a **3-tier architecture**:

1. Frontend (Client)
2. Backend (Server/API)
3. Database (Storage)
